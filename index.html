Boggle Party

🔧 One-time Firebase Setup

Firebase gives this game a live cloud database so friends on any device can play together. It's free and takes ~5 minutes to set up once.

1
Create a Firebase project

Go to console.firebase.google.com → Add project → any name → disable Analytics → Create project.

2
Create a Realtime Database

Left sidebar → Build → Realtime Database → Create database → any region → select "Start in test mode" → Enable.

3
Get your config

⚙️ Project Settings (gear icon, top-left) → scroll to "Your apps" → click </> Web → Register app → copy the firebaseConfig shown.

4
Edit this file

Open boggle-party.html in Notepad or any text editor. Find the section marked PASTE YOUR FIREBASE CONFIG HERE and replace the placeholder values with your real ones:

apiKey: "AIzaSyXXXXXXXXXXXX", authDomain: "my-game.firebaseapp.com", databaseURL: "https://my-game-default-rtdb.firebaseio.com", projectId: "my-game", storageBucket: "my-game.appspot.com", messagingSenderId: "123456789", appId: "1:123:web:abc"
5
Re-upload to Netlify

app.netlify.com → your site → Deploys tab → drag the updated file onto the page. Done! 🎉
